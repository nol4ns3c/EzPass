 Simple password generator with random words converted to leet. 


## For installation: 

``` git clone https://github.com/nol4ns3c/EzPass```

```cd EzPass```

```pip install -r requeiements.txt```



## Usage:

```python3 EzPass.py -l [Password Length]```

```Example Usage:```

```python3 EzPass.py -l 8```
